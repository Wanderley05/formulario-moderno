## Formulário Moderno

Este projeto foi desenvolvido como atividade da disciplina de **Desenvolvimento Front-End**.
A proposta da atividade foi criar um formulário de cadastro de usuários utilizando **HTML, CSS e JavaScript**, aplicando validação em tempo real e organizando o projeto no GitHub.

## Objetivo

Criar um formulário moderno e funcional, com validações que ajudam o usuário a preencher os dados corretamente e com feedback visual para facilitar o uso.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub

## Funcionalidades implementadas

* Formulário de cadastro com os campos:

  * Nome completo
  * E-mail
  * Senha
  * Confirmação de senha

* Validação dos campos em tempo real

* Mensagens de erro exibidas abaixo de cada campo

* Mudança de cor dos campos quando estão válidos ou inválidos

* Barra de força da senha

* Verificação se as senhas digitadas são iguais

* Simulação de envio do formulário com animação de carregamento

* Layout responsivo para funcionar em telas menores
